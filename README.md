# Siddhanath Yoga Marathon

A comprehensive web application for the Siddhanath Yoga Marathon - Journey Through Sacred Teachings.

## About

This application provides a structured learning path through the sacred teachings of Yogiraj Siddhanath, featuring:

- **280 Guided Sessions**: Embedded YouTube videos with progress tracking
- **Sacred Sayings Database**: 4,941 categorized spiritual quotes
- **Progress Tracking**: Personal journey tracking with streaks and achievements
- **Responsive Design**: Beautiful glass-morphism UI that works on all devices

## Features

### Guided Sessions
- 280 unique guided sessions (duplicates removed)
- Session completion tracking
- YouTube video integration

### Sacred Sayings
- 4,941 sacred sayings from Gurunath's teachings
- Search and category filtering
- Favorites system
- Timestamped content from video transcripts

### Interactive Features
- Session modals with YouTube integration
- Progress celebration animations
- Streak tracking and achievements
- Local storage for progress persistence
- Share progress functionality

## Technical Details

### Built With
- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **Styling**: Glass-morphism design with CSS Grid/Flexbox
- **Data**: JSON-based data structure
- **Video Integration**: YouTube embed API
- **Storage**: LocalStorage for progress persistence

## Data Overview

- **Total Videos**: 1,085 analyzed
- **Guided Sessions**: 280 unique sessions
- **Sacred Sayings**: 4,941 categorized quotes
- **Categories**: 6 spiritual wisdom categories

## Usage

1. **Start Your Journey**: Browse the 280 guided sessions
2. **Complete Sessions**: Watch videos and mark them complete
3. **Track Progress**: Monitor your advancement
4. **Explore Sayings**: Search and favorite sacred teachings
5. **Share Progress**: Share your spiritual journey with others

## Local Development

1. Clone the repository
2. Open `index.html` in a web browser
3. Or serve with a local HTTP server:
   ```bash
   python3 -m http.server 8000
   ```
